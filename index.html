    Creating Brands

    A brief explanation on how to create new brands…

    {
  "method": "POST",
  "summary": "Creates a new brand in the Product master API",
  "notes": "",
  "type": "Brand_topElement",
  "nickname": "createBrands",
  "produces": [
    "application/json"
  ],
  "parameters": [
    {
      "name": "brands",
      "description": "Array of brand objects",
      "type": "Brand_topElement",
      "required": false,
      "paramType": "body"
    }
  ],
  "responseMessages": [
    {
      "code": 400,
      "message": "invalid input"
    }
  ],
  "path": "/brands"
}

    Updating Brands

    A brief explanation on how to update brands…

    {
  "method": "PUT",
  "summary": "Updates an AGCO brand in the Product master API",
  "notes": "",
  "type": "Brand_topElement",
  "nickname": "updateBrands",
  "produces": [
    "application/json"
  ],
  "parameters": [
    {
      "name": "id",
      "description": "ID of the brand that needs to be updated",
      "type": "string",
      "required": true,
      "paramType": "path"
    },
    {
      "name": "Brands",
      "description": "Array of brand objects",
      "type": "Brand_topElement",
      "required": false,
      "paramType": "body"
    }
  ],
  "responseMessages": [
    {
      "code": 404,
      "message": "brand not found"
    },
    {
      "code": 400,
      "message": "invalid input"
    }
  ],
  "path": "/brands/{id}"
}
